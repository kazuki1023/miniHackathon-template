- 必要な情報は以下

| 項目 |
|--------|
| 最初の管理者の名前 |
| サークル名 |
| その人のメールアドレス |
| password |

### 終了条件
- [ ] 登録したら、今入力した人のmail宛にmailが送られる

- [ ] mailを踏むと、登録が完了し、ログインした状態でtopページに行く
